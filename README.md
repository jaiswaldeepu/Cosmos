# Cosmos
So, I made this project for the space submission. it is what as it sounds like, just a space in 3d, with sun, planets, comets, nebula, asteroids, comets, meteors, Satellites and all. but not just for illustrative purpose, it actually shows it's data when You click them, which is all taken from NASA.
Just click on whatever body you are interested in and get the data shown o. plus there is a BODIES option which could be more easier to use and find, i didn't name them, for extra curiousity.
